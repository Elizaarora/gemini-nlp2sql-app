# 🎓 NLQ — Natural Language to SQL Query App

A web application that lets you query a student database using **plain English**. Powered by **Google Gemini AI**, it converts your question into a SQL query and returns the results instantly — no SQL knowledge required.

---

## 🚀 Demo

> Type: *"Show me all students in the AI class"*
> App runs: `SELECT * FROM STUDENT WHERE CLASS = "AI";`
> And displays the matching student records.

---

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| [Streamlit](https://streamlit.io/) | Web UI framework |
| [Google Gemini AI](https://aistudio.google.com/) | Converts English → SQL |
| [SQLite](https://www.sqlite.org/) | Lightweight local database |
| [python-dotenv](https://pypi.org/project/python-dotenv/) | Manages secret API keys |

## ⚙️ Setup & Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/nlq.git
cd nlq
```

### 2. Create a Virtual Environment

```bash
python -m venv venv
```

### 3. Activate the Virtual Environment

**Windows:**
```bash
venv\Scripts\activate
```

**Mac/Linux:**
```bash
source venv/bin/activate
```

You'll see `(venv)` appear in your terminal — that means it's active.

### 4. Install Dependencies

```bash
pip install -r requirements.txt
```

### 5. Get Your Google Gemini API Key

1. Go to [https://aistudio.google.com/app/apikey](https://aistudio.google.com/app/apikey)
2. Create a new API key
3. Copy it

### 6. Create the `.env` File

In the project root folder, create a file named `.env` and add:

```
GOOGLE_API_KEY=your_actual_api_key_here
```

> ⚠️ Never share this file or push it to GitHub. It's already listed in `.gitignore`.

### 7. Create the Database (Run Once)

```bash
python sql.py
```

This creates `student.db` with sample student records.

### 8. Run the App

```bash
streamlit run app.py
```

Your browser will open automatically at `http://localhost:8501`

## 🔍 How It Works

```
User types question
       ↓
Gemini AI converts it to SQL
       ↓
SQL runs on student.db
       ↓
Results displayed on screen
```

1. You type a plain English question in the text box
2. The app sends your question + database context to **Google Gemini**
3. Gemini returns a valid SQL query
4. The app executes that query on `student.db`
5. Results are displayed in the browser

## ⚠️ Common Issues

| Problem | Fix |
|---|---|
| `ModuleNotFoundError: No module named 'google.protobuf'` | Make sure venv is activated and run `pip install -r requirements.txt` inside it |
| `WARNING: Package(s) not found: streamlit` | Your venv doesn't have streamlit — run `pip install streamlit` |
| `GOOGLE_API_KEY` not working | Check your `.env` file exists in the project root with the correct key |
| Duplicate records in DB | Open `sql.py`, uncomment `cursor.execute("DELETE FROM STUDENT;")` and run it once |
